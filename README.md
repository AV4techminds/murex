🚀 AV4FintechMinds

Vision

AV4FintechMinds is a Banking and Financial Technology learning project inspired by Murex architecture. The goal of this project is to gain practical knowledge in Murex concepts, Python automation, Oracle database development, DevOps fundamentals, workflow orchestration, and regulatory reporting through a simulated banking trade processing platform.

🏦 What is Murex?

Murex is a leading capital markets platform used by global banks and financial institutions for trade processing, risk management, P&L calculation, settlement, accounting, and regulatory reporting.

Murex provides a centralized platform where Front Office, Middle Office, Back Office, Risk, Finance, and Regulatory teams work using a single source of truth.
